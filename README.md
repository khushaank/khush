## 🚀 Getting Started

### Prerequisites

- A web browser
- (Optional) Live Server extension for local development

### Local Development

1. Clone the repository:

```bash
git clone https://github.com/khushaank/khush.git
cd khush
```

2. Open with Live Server or any local web server:

```bash
# If you have Python installed:
python -m http.server 8000

# Or use VS Code Live Server extension
```

3. Navigate to `http://localhost:8000` (or your server's URL)

### Environment Setup

The site uses Supabase for backend services. The configuration is currently embedded in the JavaScript files for demonstration purposes.
